# 55315B Introduction to SQL Databases (2025)
Courseware Access:  https://www.softwaretutorialservices.com/course/55315b

This three-day instructor-led course is aimed at people looking to move into a database professional role or whose job role is expanding to encompass database elements. The course describes fundamental database concepts including database types, database languages, and database designs, with a focus on SQL Server 2025 and SSMS 22. Students gain hands-on experience exploring, querying, designing, and managing SQL Server databases using industry-standard techniques.

Audience:<br>
The primary audience for this course is those just starting in a database role, or whose role has expanded to include database technologies. Developers that deliver content from SQL Server databases will also benefit from this material.

After completing this course, students will be able to:<br>
* Describe and apply key relational database concepts in SQL Server 2025, including database types, the ANSI/SPARC architecture, SQL language categories, and new SQL Server 2025 and SSMS 22 features
* Design effi cient relational database schemas by applying entity relationship diagramming, normalization rules through Third Normal Form, and selective denormalization strategies
* Implement and enforce data integrity across related tables in SQL Server 2025 using foreign key constraints with cascading options, DML triggers, and DDL triggers
* Evaluate and improve database performance by designing effective indexing strategies, rewriting nonsargable queries, and understanding the effects of transaction isolation levels and concurrency control
* Create and use essential SQL Server 2025 database objects — including tables with constraints, standard and indexed views, stored procedures with parameters, DMLand DDL triggers, and scalar and table-valued user-defined functions — to meet theneeds of database administrators and application users
